#각 쿼리 타입 설명 및 사용법

#TYPE 1
설명: 'Mapo' 구역에서 판매 중인 집의 주소를 찾습니다.
사용법:
1을 입력하여 TYPE 1을 선택합니다.
서브타입 메뉴에서 1을 입력하여 'Mapo' 구역에서 가격이 ￦1,000,000,000에서 
￦1,500,000,000 사이인 부동산의 주소를 찾습니다.

#TYPE 2
설명: 8학군에 위치한 부동산의 주소를 찾습니다.
사용법:
2를 입력하여 TYPE 2를 선택합니다.
서브타입 메뉴에서 1을 입력하여 8학군에서 침실이 4개 이상이고 욕실이 2개인 부동산의 주소를 찾습니다.

#TYPE 3
설명: 2022년에 가장 많은 판매액을 기록한 에이전트를 찾습니다.
사용법:
3을 입력하여 TYPE 3을 선택합니다.
서브타입 메뉴에서 1을 입력하여 2023년에 가장 많은 판매액을 기록한 상위 k명의 에이전트를 찾습니다. k값을 입력합니다.
서브타입 메뉴에서 2를 입력하여 2021년에 가장 낮은 판매액을 기록한 하위 10%의 에이전트를 찾습니다.

#TYPE 4
설명: 2022년에 각 에이전트의 평균 판매 가격과 평균 시장에 머문 기간을 계산합니다.
사용법:
4를 입력하여 TYPE 4를 선택합니다.
서브타입 메뉴에서 1을 입력하여 2023년 동안 각 에이전트가 판매한 부동산 중 최고 가격을 계산합니다.
서브타입 메뉴에서 2를 입력하여 각 에이전트가 판매한 부동산 중 시장에 가장 오래 머문 기간을 계산합니다.

#TYPE 5
설명: 각 부동산 유형별로 가장 비싼 부동산의 사진 URL을 표시합니다.
사용법:
5를 입력하여 TYPE 5를 선택합니다.

#TYPE 6
설명: 새로운 부동산 거래를 기록합니다.
사용법:
6을 입력하여 TYPE 6을 선택합니다.
거래 세부 정보를 입력합니다 (거래 ID, 부동산 ID, 구매자 ID, 판매 에이전트 ID, 구매 에이전트 ID, 판매 가격, 판매 날짜).

#TYPE 7
설명: 새로운 에이전트를 추가합니다.
사용법:
7을 입력하여 TYPE 7을 선택합니다.
에이전트 세부 정보를 입력합니다 (에이전트 ID, 이름, 전화번호).
