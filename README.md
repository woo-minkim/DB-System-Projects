# DB System Projects

부동산 거래 도메인을 대상으로 **데이터 모델링부터 MySQL 물리 스키마 구현, C++ 기반 질의 프로그램 개발까지 수행한 데이터베이스 프로젝트 모음**입니다.  
도메인 요구사항을 관계형 데이터 구조로 설계하고 이를 실제 DB와 애플리케이션 로직으로 연결하는 전 과정을 프로젝트 2단계로 나누어 구현했습니다.

## Overview

이 저장소는 다음 흐름을 중심으로 구성되어 있습니다.

- 부동산 거래 도메인 요구사항 분석
- ER 기반 개념/논리 모델 설계
- MySQL 물리 스키마 및 샘플 데이터 구현
- C++ 콘솔 프로그램을 통한 조회, 집계, 입력 기능 검증
